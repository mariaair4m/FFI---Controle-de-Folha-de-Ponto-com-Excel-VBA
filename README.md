# FFI---Controle-de-Folha-de-Ponto-com-Excel-VBA
Sistema desenvolvido em Microsoft Excel com VBA para automatizar o controle de frequência e folha de ponto. O projeto utiliza macros para agilizar o preenchimento, organização e consulta de informações, reduzindo processos manuais e aumentando a eficiência na gestão dos registros.
