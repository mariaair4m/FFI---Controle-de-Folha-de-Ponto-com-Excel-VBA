# FFI---Controle-de-Folha-de-Ponto-com-Excel-VBA
Sistema desenvolvido em Microsoft Excel com VBA para automatizar o controle de frequência e folha de ponto. 


## 📋 Sobre o Projeto
O sistema permite gerenciar registros de frequência por meio de planilhas automatizadas. O projeto utiliza macros para agilizar o preenchimento, organização e consulta de informações, reduzindo processos manuais e aumentando a eficiência na gestão dos registros.


## ⚙️ Funcionalidades
- Cadastro de funcionários atraves de planilhas
- Criação de planilhas personalizadas
- Automatização de impressão
- Sistema de impressão
- Controle de frequência
- Implementação de fins de semana e feriados atomáticos


## 🛠️ Tecnologias Utilizadas
- Microsoft Excel
- VBA (Visual Basic for Applications)


## 🚀 Como Utilizar o sistema
### 1. Abrindo o Arquivo
1. Baixe o seguinte arquivo: [FFI](ffi.xls)
2. Abra o arquivo utilizando o Microsoft Excel.
3. Caso apareça um aviso de segurança, clique em **Habilitar Conteúdo** para permitir a execução das macros.


### 2. Ativando as Macros
Ao abrir a planilha, o Excel pode bloquear automaticamente a execução de macros.

Para habilitá-las:

1. Clique em **Habilitar Edição** (se exibido).
2. Clique em **Habilitar Conteúdo** na barra amarela de segurança.
3. Aguarde o carregamento completo do sistema.

> As funcionalidades do projeto dependem da execução das macros VBA.


### 3. Menu "Gerar Folha de Frequência"
O menu **Gerar Folha de Ponto** é responsável pela criação automática das folhas de frequência.

Funções disponíveis:

- Seleção do mês e ano para atualização de calendário.
- Seleção de planilha de funcionario.
- Geração automática da folha de ponto.
- Preenchimento e organização dos registros.
- Atualização das informações necessárias para controle de frequência.


### 4. Menu "Impressão"
O menu **Impressão** foi desenvolvido para facilitar a emissão das folhas geradas.

Funções disponíveis:

- Conexão manual pelo VBA paar conexão com impressora.
- Opções de escolha de planilhas especificas para impressão.
- Impressão das folhas de ponto.
  

>   5. Observações

>- É necessário utilizar o Microsoft Excel com suporte a macros.
>- O sistema foi desenvolvido utilizando VBA (Visual Basic for Applications).
>- Os dados presentes nesta versão foram anonimizados para fins de demonstração e portfólio.

Durante o desenvolvimento foram utilizados:
